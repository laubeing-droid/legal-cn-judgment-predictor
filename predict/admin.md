# 行政判决预测

## 预测维度

### 一、行政行为类型识别
- 行政处罚 / 行政许可 / 行政强制 / 行政征收 / 行政协议 / 行政不作为

### 二、合法性审查
使用行政行为合法性审查框架：
1. **职权**：行政机关是否具有法定职权
2. **事实**：事实认定是否清楚、证据是否充分
3. **程序**：是否履行法定程序（告知/听证/送达）
4. **裁量**：行政裁量是否适当（是否明显不当）

### 三、判决方向预测
| 结果 | 条件 |
|:--:|:-----|
| 🟢 维持 | 行政行为合法 |
| 🟡 确认违法但不撤销 | 违法但撤销会损害公共利益 |
| 🟠 撤销/部分撤销 | 主要证据不足/适用法律错误/程序违法/超越职权/滥用职权 |
| 🔴 变更 | 行政处罚明显不当 |
| ⬛ 确认无效 | 行政行为重大且明显违法 |

### 四、关键因子
- 程序瑕疵严重程度
- 事实认定是否可补正
- 裁量是否超出合理范围
