# 民事判决预测

## 预测维度

### 一、案由确认
基于事实要素抽取结果，确定精确案由：
- 参考：民法典合同编/物权编/侵权责任编/婚姻家庭编/继承编
- 引用：law-semantic-tree.json 对应法条→要件→效果

### 二、请求权基础分析
使用请求权基础分析法（非 IRAC）：
1. **请求权目标**：原告主张的权利内容
2. **请求权基础**：支持该权利的法条（精确到款/项）
3. **构成要件检视**：逐项对照法条要件是否满足
4. **抗辩事由检视**：被告抗辩是否成立
5. **法律效果**：请求权的具体法律效果

### 三、胜败方向预测
| 等级 | 说明 |
|:--:|:-----|
| 🟢 极可能胜诉 | 请求权基础明确 + 构成要件全部满足 + 抗辩不成立 + 证据充分 |
| 🟡 较可能胜诉 | 请求权基础明确 + 要件基本满足 + 部分抗辩可能成立 |
| 🟠 结果不确定 | 关键事实存疑 + 法条适用有争议空间 + 取决于证据补充 |
| 🔴 较可能败诉 | 请求权基础存疑 + 构成要件不完全满足 + 抗辩可能成立 |
| ⬛ 极可能败诉 | 请求权基础不成立 / 时效已过 / 构成要件明显不满足 |

### 四、金额预测
- 计算方法：逐项列出计算公式 + 法条依据
- 金额区间：[下限] - [上限]
- 关键变量：违约责任比例 / 过失相抵 / 减损义务

### 五、关键因子
- 对原告有利的因子
- 对被告有利的因子
- 不确定因子（需补充证据/取决于法官自由裁量）
