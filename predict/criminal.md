# 刑事判决预测

## 预测维度

### 一、罪名确定
基于犯罪构成要件分析（三阶层或四要件）：
- 四要件：客体/客观方面/主体/主观方面
- 三阶层：构成要件该当性/违法性/有责性
- 引用 law-semantic-tree.json 刑法分则对应罪名→要件→量刑

### 二、定罪可能性
| 等级 | 说明 |
|:--:|:-----|
| 🟢 极可能定罪 | 全部构成要件满足 + 无出罪事由 |
| 🟡 较可能定罪 | 构成要件基本满足 + 部分辩护可成立但不足以出罪 |
| 🟠 存疑 | 关键构成要件证据不足 / 罪名认定存在争议 |
| 🔴 较可能无罪 | 构成要件不完全满足 / 存在正当化事由 |
| ⬛ 极可能无罪 | 行为不构成犯罪 / 事实不清证据不足 |

### 三、量刑预测
- **基准刑**：根据刑法分则 + 量刑指导意见确定
- **从重情节**：累犯/主犯/特定时期/特定对象
- **从轻/减轻情节**：自首/立功/从犯/认罪认罚/退赃退赔
- **刑期区间**：[下限] - [上限]
- **缓刑可能性**：是否满足缓刑条件（刑法第72条）
